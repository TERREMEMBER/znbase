The ZNbaseDB release that is powering a universe can be upgraded to get the new features and fixes in a release.

A rolling upgrade can be performed on a live universe deployment by following these steps:

1. Go to the **Universe Detail** page.
2. From the **More** drop-down list, select **Upgrade Software**.
3. In the confirmation dialog, select the new ZNbaseDB version from the drop-down list. The ZNbase Platform console will upgrade the universe in a rolling manner.
