ZNbase Cloud is a fully-managed ZNbaseDB-as-a-Service that allows you to run ZNbaseDB clusters on
<a href="https://cloud.google.com/">Google Cloud Platform (GCP)</a>, 
<a href="https://aws.amazon.com/">Amazon Web Services (AWS)</a>, and
<a href="https://azure.microsoft.com/">Microsoft Azure</a> (coming soon). You can use the Free Tier level to get started with ZNbaseDB. Although
not suitable for production workloads, the Free Tier includes enough resources to start exploring the core features available for developing applications with ZNbaseDB.
