Resources to learn more about distributed SQL and building applications on top of ZNbaseDB. This directory contains markdown files that are rendered in the [Learn ZNbaseDB](https://learn.ZNbase.com) site.

If you are looking to run in development go to [https://github.com/ZNbase/learn-ZNbase](https://github.com/ZNbase/learn-ZNbase) and follow the README there.

## File Structure
We separate the content into chapters as **.md** files and group the chapters into a lesson denoted by a parent directory. A group of chapters will form a series that are top-level directories.
